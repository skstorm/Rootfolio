# 루트폴리오 (Rootfolio) 🌳

> **"성장을 위한 뿌리, 성취를 위한 포트폴리오"**

루트폴리오는 학습 여정, 개발 프로젝트, 개인적인 성찰을 하나의 응집된 허브로 통합하기 위해 설계된 전문적인 디지털 생태계입니다.

## 🚀 개요

이 저장소는 다음을 위한 중앙 관제 센터 역할을 합니다:
- **지식 베이스 (Knowledge Base)**: 기술, 과학 및 개인 학습에 대한 심도 있는 탐구.
- **앱 쇼케이스 (App Showcase)**: 허브에 직접 통합된 고성능 애플리케이션 (Flutter, Web).
- **데일리 로그 (Daily Logs)**: 시간 흐름에 따른 성장을 추적하기 위한 성찰적인 일기 기록.

## 🏗️ 아키텍처

이 프로젝트는 확장성을 극대화하기 위해 모듈형 섹션으로 구성되어 있습니다:

- `platform/`: 핵심 UI 및 내비게이션 엔진 (Vite + Vanilla JS).
- `content/`: 
    - `studies/`: 마크다운 기반의 학습 노트.
    - `logs/`: 개인적인 성찰 및 데일리 로그.
- `modules/apps/`: 독립적인 대화형 애플리케이션 및 도구들.

## 🛠️ 기술 스택

- **코어 허브 (Core Hub)**: HTML5, CSS3 (Modern System), Vanilla JavaScript.
- **빌드 도구 (Build Tool)**: Vite.
- **통합 앱 (Integrated Apps)**: Flutter (Web/Mobile) 등.
- **스타일링 (Styling)**: 프리미엄 다크 테마, 반응형 디자인.
