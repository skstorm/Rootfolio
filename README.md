# Rootfolio 🌳

> **"Roots for Growth, Portfolio for Achievement"**

Rootfolio is a professional digital ecosystem designed to unify learning journeys, development projects, and personal reflections into a single, cohesive hub.

## 🚀 Overview

This repository serves as the central command center for:
- **Knowledge Base**: Deep dives into tech, science, and personal studies.
- **App Showcase**: High-performance applications (Flutter, Web) integrated directly into the hub.
- **Daily Logs**: Reflective diary entries to track growth over time.

## 🏗️ Architecture

The project is organized into modular sections for maximum scalability:

- `platform/`: The core UI and navigation engine (Vite + Vanilla JS).
- `content/`: 
    - `studies/`: Markdown-driven learning notes.
    - `logs/`: Personal reflections and daily logs.
- `modules/apps/`: Independent interactive applications and tools.

## 🛠️ Tech Stack

- **Core Hub**: HTML5, CSS3 (Modern System), Vanilla JavaScript.
- **Build Tool**: Vite.
- **Integrated Apps**: Flutter (Web/Mobile), etc.
- **Styling**: Premium Dark Theme, Responsive Design.
