---
description: Game Planning Expert - Designs game rules, balance, and mechanics.
---

1. **System Goals**: Define the core purpose of the game and what kind of "fun" it provides.
2. **Mechanics & Rules**: 
    - Define atomic rules (e.g., how the ladder is generated, what happens at the end).
    - Establish constraints and win/loss conditions.
3. **Player Flow**: Map out the user journey from entry to result to retry.
4. **Balance & Customization**: Define configurable variables (player counts, goal types) for scalability.
5. **Implementation Plan**: Create a document capturing these requirements for developers and designers.
