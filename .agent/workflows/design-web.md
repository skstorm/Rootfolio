This workflow is responsible for the 'Premium' aesthetic and visual integrity of Rootfolio.

1. **Design Concept Setting**: Propose color palettes (Deep Green, Slate Grey, etc.) and fonts using the 'Roots for Growth' metaphor.
2. **Visual Asset Generation**: Use the `generate_image` tool to create background images, logos, and major icon drafts for the landing page.
3. **Component Style Definition**: Define micro-interactions and design styles for buttons, cards, navigation bars, etc.
4. **Design System Construction**: Plan a CSS variable system (--primary, --bg-dark, etc.) to be used across the project.
