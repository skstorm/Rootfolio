description: Web Planning Expert - Designs the structure and requirements of the project.
---

This workflow focuses on designing the information architecture and user experience of Rootfolio.

1. **Status Analysis**: Analyze the current `README.md` and file structure to set the baseline for planning.
2. **Requirement Definition**: 
    - Define detailed features needed for the Knowledge Base, App Showcase, and Daily Logs.
    - Create user scenarios to define what experience visitors should have on the site.
3. **Site Map Creation**: Clarify the overall page structure and paths (/studies, /apps, /logs, etc.).
4. **Plan Generation**: Create `implementation_plan.md` to organize the planning details and request user approval.
