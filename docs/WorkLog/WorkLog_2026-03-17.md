## 🛠 트러블슈팅
### 1. `StateProvider` 메서드 누락 컴파일 오류
- **문제**: `flutter_riverpod` 버전 특성상 `StateProvider`에서 `update` 메서드를 찾지 못하는 오류 발생.
- **해결**: 프로젝트 내 다른 프로바이더와 동일한 `Notifier` 기반의 `debugEnabledProvider`로 변경하여 안정적인 상태 변경 로직 확보.

### 2. 가이드라인(빨간 사각형) 토글 미연동
- **문제**: 디버그 UI를 꺼도 `ScratchPainter` 내부의 텍스트 영역 빨간 사각형이 사라지지 않음.
- **해결**: `ScratchPainter`에 주입된 `debugService.isDebugMode` 상태를 전파하여 사각형 렌더링을 조건부로 수정.

### 3. 멀티 플랫폼(PC/모바일) 동기화 지연
- **문제**: 프로바이더 구조 변경 후 핫 리로드만으로는 PC와 갤럭시 단말기에 버튼이 즉시 나타나지 않음.
- **해결**: 양쪽 플랫폼에 **핫 리스타트(Hot Restart)**를 강제 실행하여 전역 상태 및 UI 트리를 최신화함.
