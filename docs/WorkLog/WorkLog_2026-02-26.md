## 🛠️ 트러블슈팅 (Troubleshooting)
- **Flutter 설치 패키지 충돌**: `winget`을 통한 설치 시 패키지명 모호성으로 오류 발생.
    - **해결**: 더욱 안정적인 버전 관리 도구인 `puro`를 도입하여 Flutter SDK 설치 및 관리 환경 개선.
- **배포 스크립트 실행 오류**: ESM 환경에서의 순환 참조 및 `ts-node` 경로 설정 문제로 배포 자동화 스크립트 중단.
    - **해결**: 수동 빌드(`flutter build web`) 및 파일 이동 프로세스를 수행하여 즉시 대응 후, 플랫폼 연동성 확보.
- **Iframe 내 리소스 로딩 실패 (404)**: Flutter 앱이 Rootfolio iframe 내에서 로드될 때 절대 경로 참조 이슈 발생.
    - **해결**: `index.html`의 `<base href="/">`를 `<base href="./">`로 수정하여 하위 디렉토리 배포 환경에 최적화.
- **반복적인 명령어 승인 절차**: 잦은 명령어 실행 시 사용자 피로도 증가.
    - **해결**: 워크플로우 파일에 `// turbo-all` 주석을 도입하고 자동 실행 모드를 활성화하여 개발 생산성 극대화.
