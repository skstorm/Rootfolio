## 🔥 트러블슈팅

### 1. `build_runner` 0-output 및 크래시
- **증상**: `injectable_generator`가 코드를 생성하지 못하고 내부 크래시 발생
- **해결**: `pubspec.yaml` 정비 및 환경 설정(env) 단순화 시도 후, 최종적으로 **수동 DI 등록(`manual_setup.dart`)**으로 우회하여 개발 중단 방지

### 2. Gemini API 모델 명칭 및 버전 불일치
- **증상**: `gemini-1.5-flash` 모델이 `v1beta`에서 찾을 수 없거나 지원하지 않는다는 오류 발생
- **분석**: `curl`을 통한 모델 리스트 직접 조회 결과, API 키 권한 및 버전 매칭 이슈 확인
- **시도**: `gemini-1.5-flash-latest`, `gemini-2.0-flash` 등 다양한 모델로 교체 테스트 진행

### 3. 무한 로딩 페이지 처리
- **증상**: AI 호출 실패 시 UI가 로딩 상태에서 멈춤
- **해결**: `TitleNotifier`에 전역 `try-catch`를 적용하여 에러 발생 시 즉시 사용자에게 메시지 표시 및 재시도 버튼 활성화
