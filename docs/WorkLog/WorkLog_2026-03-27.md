## 🔥 트러블슈팅

### 앱 기동 시 화이트스크린(White Screen) 발생
- **원인**: `lib/di/injection_container.dart`에서 `getIt.init()`을 호출할 때 `await`가 누락됨. 비동기 의존성 등록이 완료되기 전에 `main()`이 다음 단계 (`getIt<AppRuntimeConfig>()` 참조 등)로 넘어가며 크래시 발생.
- **해결**: `await getIt.init(environment: env);` 구문으로 수정하여 해결.
- **교훈**: 비동기 처리가 포함된 DI 초기화 단계는 반드시 동기적으로 완결됨을 보장해야 함.

### 유닛 테스트 빌드 오류 (`AdRewardResult` 생성자)
- **원인**: 테스트 코드 내에서 `AdRewardResult` 객체 생성 시 존재하지 않는 `isRewarded` 파라미터를 사용하려 시도함.
- **해결**: 실제 클래스 정의에 맞게 `status: AdRewardStatus.rewarded` 형식으로 수정 후 `build_runner` 재실행.
